# Iframe Block

Embed external websites and content into your documentation pages.

## Syntax

```markdown
<iframe src="https://www.example.com"/>
```

## Parameters

- **src**: URL of the external content to embed (required)
  - Must be a fully qualified URL (e.g., `https://www.example.com`)
  - The external site must allow iframe embedding
