# Resources Block without Hero

Resource links without hero banner.

## Syntax

```markdown
<Resources slots="heading, links"/>

#### Resources

* [Link 1](url)
* [Link 2](url)
```

## Parameters

- **slots**: `"heading, links"` - Heading and links
