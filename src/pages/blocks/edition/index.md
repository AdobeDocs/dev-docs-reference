# Edition Block

Display edition or platform labels.

## Syntax

```markdown
<Edition slots="text" />

Enterprise Edition
```

## Parameters

- **slots**: `"text"` - Edition text or link

## Examples

### Plain Text

<Edition slot="text"/>

PaaS Only

### With Link

<Edition slot="text"/>

[Enterprise Edition](https://example.com/editions)
